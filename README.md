## ⚙️ Jenkins CI/CD Pipeline

A Jenkins Declarative Pipeline was implemented to automate the deployment process.

### Pipeline Stages

1. Build
2. Test
3. Deploy

### Deployment Process

* Source code is fetched from GitHub.
* Docker image is built automatically.
* Docker container is deployed using Jenkins.
* Successful pipeline execution validates the deployment workflow.
